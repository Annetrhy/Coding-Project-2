# Product Dashboard

A lightweight web dashboard demonstrating modern JavaScript data fetching and UI rendering.

## Overview

This project fetches product data from a remote API and displays a responsive product gallery using vanilla HTML, CSS, and JavaScript.

## Features

- Fetches product data using both promise-based `.then()` and `async/await` patterns
- Renders product cards with image, name, and formatted price
- Simple responsive layout with hover interactions
- Clean separation of UI markup and data logic

## Files

- `index.html` — application structure and script inclusion
- `style.css` — responsive layout and card styling
- `cp_2.js` — API fetch logic, error handling, and dynamic rendering

## Run locally

1. Open `index.html` in a browser
2. Alternatively, serve the folder with a local HTTP server
3. The dashboard automatically loads and displays the first products from the API
